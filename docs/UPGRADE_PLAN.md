# Upgrade Plan

## Delivered MVP

- Stable email support polling.
- Draft-only Kethura growth cycle.
- Server-side operations API protected by the existing agent token.
- Admin approval dashboard and audit tables.
- Daily business report command.
- High-risk support escalation.
- World Cup 2026 campaign API, admin controls, Flutter screens, scoring, rewards, and referrals.

## Next controlled phase

1. Connect read-only business metrics to opportunity detection.
2. Configure one X account and test posting in a non-production account.
3. Add a signed action executor that only processes approved actions.
4. Add Meta Page permissions after app review.
5. Add delivery receipts and performance metrics for published content.
6. Move Kethura operational storage from JSON files to a managed database.

## Release gates

- Backup verified.
- PHP/Node/Flutter checks pass.
- Migrations run on beta only.
- Push notifications remain disabled.
- No social credentials means no public execution.
- Production rollout requires a separate approval and rollback plan.
